## Installation

1. Download the JSON file for your filament
2. Copy it to your Creality Print user filament folder:
   ```
   C:\Users\[YOUR NAME]\AppData\Roaming\Creality\Creality Print\6.0\user\[ID]\filament\
   ```
3. Restart Creality Print
4. Select the profile from the filament dropdown

## Profile Settings

| Material | Nozzle Temp | Bed Temp | Max Flow |
|---|---|---|---|
| PLA Basic | 220°C | 55°C | 16 mm³/s |
| PLA Turbo | 230°C | 60°C | 32 mm³/s |
| PETG | 240°C | 75°C | 12 mm³/s |
| PP | 230°C | 90°C | 8 mm³/s |
| PP-CF | 240°C | 95°C | 6 mm³/s |
| PA Nylon | 250°C | 80°C | 8 mm³/s |

> All profiles inherit from the corresponding Creality Generic base and require **Creality Print 6.0+**.
